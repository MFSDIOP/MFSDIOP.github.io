Présentation

Bienvenue dans le site officiel de GenieCivil2D qui est un projet réalisé par deux étudiantes en Licence 3 en Génie Informatique

En effet le GenieCivil2D permet de tracer tous vos figures avec de simples bouts de code, ainsi dans cette page vous verrez deux figures réalisé à partir de ce dernier en utilisant le module turtle de Python et le module genieCivilOuvrage2D

Maquette d'un pont

Cette figure reprèsente la maquette d'un pont tracée grâce aux fonctions Triangle, Rectangle et Ellipse de la module genieCivilOuvrage2D.

Vous pouvez voir l'image de la figure en cliquant sur le bouton suivant
